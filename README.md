# ORI-A begrippenlijsten

Nuttige begrippenlijsten binnen ORI-A die beheerd worden door de [Werkgroep Archivering Raadsinformatie](https://ori-a.nl/colofon#werkgroep-archivering-raadsinformatie). Opgesteld in [SKOS](https://en.wikipedia.org/wiki/Simple_Knowledge_Organization_System). 

De gerenderde begrippenlijsten kun je vinden op <https://ori-a.nl/begrippenlijsten>.
